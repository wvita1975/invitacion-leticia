# 🎉 Invitación Cumpleaños Leticia Vita - Versión Simple

Invitación interactiva optimizada para Vercel con React + Vite.

## 🚀 DESPLIEGUE EN VERCEL (3 PASOS)

### 1. Sube a GitHub
```bash
git init
git add .
git commit -m "Invitación cumpleaños Leticia"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/invitacion-leticia.git
git push -u origin main
```

### 2. Importa en Vercel
1. Ve a [vercel.com](https://vercel.com)
2. Clic en "New Project"
3. Importa tu repositorio
4. Clic en "Deploy"

### 3. ¡Listo!
Tu invitación estará en línea en 1-2 minutos.

## 💻 Desarrollo Local
